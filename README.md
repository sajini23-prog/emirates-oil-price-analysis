# Description: This project analyses Brent crude oil prices to understand trends affecting airline fuel costs.
# Tools Used: Python , Excel , Linear Regression
# Key Tasks: Data cleaning and preprocessing , Time series analysis using moving averages , Model evaluation (R², RMSE, MAE)
# Outcome: Developed a predictive model to support fuel cost forecasting and business decision-making.
